# 🌊 Ocean Dive — Premium Diving Experiences

> Explore the deep sea and discover incredible marine life hidden beneath the waves.

A modern, responsive website for **Ocean Dive**, a premium diving experiences company. Built with semantic HTML5, modern CSS3, and vanilla JavaScript.

---

## ✨ Features

### 🎨 Design System
- **CSS Variables** — Centralized design tokens for colors, typography, spacing
- **Google Fonts** — Inter + Outfit for premium typography
- **Glassmorphism** — Frosted glass effects on cards, navbar, and sections
- **Custom Scrollbar** — Gradient-styled browser scrollbar
- **Dark Theme** — Deep blue/cyan/violet color palette

### ⚡ Animations & Interactions
- **Particle Background** — Animated ocean particles using HTML5 Canvas
- **Scroll Reveal** — Elements animate into view with staggered delays
- **Sticky Navbar** — Glassmorphism navbar appears on scroll
- **Parallax Hero** — Subtle parallax effect on hero image
- **Animated Counters** — Numbers count up when stats section is visible
- **Testimonials Carousel** — Auto-playing review slider with dot navigation
- **Preloader** — Animated loading screen with gradient bar

### 📱 Responsive Design
- **4 Breakpoints** — Desktop, tablet, mobile, and small mobile
- **Hamburger Menu** — Animated mobile navigation with overlay
- **Touch Support** — Optimized for touch devices
- **Accessibility** — Focus-visible outlines, reduced-motion support, ARIA labels

### 📄 Pages
| Page | Description |
|------|-------------|
| `indexmar.html` | Homepage — Hero, cards, stats, features, testimonials, interview, video, newsletter |
| `catalogo.html` | Catalog — Search bar, category filters, product grid with badges and pricing |

## 🎯 Key Sections

### Homepage
- **Hero** — Full-viewport landing with gradient background and floating image

## ♿ Accessibility

- Semantic HTML5 elements (`header`, `main`, `nav`, `section`, `footer`)
- ARIA labels on interactive elements
- Focus-visible keyboard navigation outlines
- `prefers-reduced-motion` support for users who prefer less animation
- High contrast text on dark backgrounds
